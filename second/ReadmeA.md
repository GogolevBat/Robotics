# Для запуска программы
## Нужно установить зависимости 

```shell
pip install -r requirements.txt 
```

## Запуск

```shell
python /second/moduleA.py
```

# Работа в программе

## Статистика
На верху формы можно увидеть таблицы:
Над ними состояние дублирующее светофор

Маленькая таблица - демонстрирует положение инструмента и его рабочее состояние
Таблица большая соответствует статусу всех моторов робота, их поворота в разных измерениях, и температуру

Слева интерфейса распологается поле для вывода внутренней информации программы (логов), там можно посмотреть все ли идет как заплонированно

## Управление
### Основные кнопки
Внизу интерфейса распологаются кнопки

Включить - Запускает моторы робота 
Выключить - Останавливает моторы робота
Пауза - Останавливает моторы робота
Начальная позиция - Устанавливает роботу начальное положение
Аварийная остановка - Останавливает работу моторов и конвейера

### Ручное управление 
Нужно выбрать страницу "Ручное управление"

Слева окна реализованно управление "джойстиками", два окна линейное управление и по суставное

Также работа с захватом и отхватом объекта, с состоянием перемещенных объектов

И работа с конвейером запуск и остановка
