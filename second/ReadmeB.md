# Работа с автоматическим управлением
По середине кнопки для работы с установкой действий алгоритма:

1) Выбор типа объекта (1-4(А-Г) и Брак)
2) Удаление последней записи и полная очистка алгоритма
3) Начало выполнения

Слева распологается визуализация текущего алгоритма:

1) Таблица визализации
2) Кнопки для сохранения и загрузки файла

Справа состояние паллетта

Состояние текущей загруженности палеты
Кнопка "новая палета" отчищает состояние загруженности
Кнопка "Очистить состояние" отчищает кол-во перенесенных данных
