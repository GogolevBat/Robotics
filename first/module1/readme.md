# Запуск приложения 
заходим в интерфейс устанавливаем все зависимости в моем случае через uv, при необходимости найти в интернете

```shell
uv sync --frozen
```
для запуска приложения используем команду, убедитесь что вы назодитесь в той же директории что и файл
```shell
uv run project.py
```
Все команды выполнять в терминале, желательно IDE
